# EM27_ICONART_preprocessing
Python scripts to preprocess and compare EM27 L2 XCO2 observations with ICON-ART model output.
## Features
- Horizontal interpolation of model data to observation site.
- Vertical interpolation to AVK heights, with optional linear extrapolation.
- Simple hourly averaging of observations for comparison with instantaneous model output.
- Algorithm's flow chart illustrated in `diag.pdf`.
